## これは何か

**SDI（球面逸脱指数 / Spherical Deviation Index）** は、  
労働・投資・公務・議員活動における報酬が、実体貢献とどれだけ乖離しているかを  
客観的に測定する指標です。

$$\text{SDI} = \frac{\text{利益}}{\text{実体貢献} \times \sqrt{T} \times \text{社会修復係数}}$$

- SDI ≦ 1.0 → 球面内・均衡（正当）
- SDI > 1.0 → 球面外・逸脱（不均衡）

感情論ではなく、**構造として**不公正を見る。  
それがこのダッシュボードの目的です。
